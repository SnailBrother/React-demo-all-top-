可视化布局结构
┌─────────────────────────────────────────┐
│                Header                    │  ← 条件显示 (有侧边栏时才显示)
├─────────────┬───────────────────────────┤
│             │          Tabs             │  ← 条件显示
│   Sidebar   ├───────────────────────────┤
│             │                           │
│             │       mainContent         │  ← 主要内容区域
│             │          {children}       │
│             │                           │
└─────────────┴───────────────────────────┘
│                                         │
│           BottomNav                     │  ← 条件显示 (底部导航模式)
└─────────────────────────────────────────┘

1. 侧边栏布局 (桌面端)
Header (显示)
┌─Sidebar ─ Content Area ─┐
│                         │
│    Tabs (显示)          │
│                         │
│    mainContent          │
│                         │
└─────────────────────────┘
BottomNav (不显示)

2. 底部导航布局 (移动端)

Header (不显示)
┌─────────Content Area────────┐
│                             │
│    Tabs (不显示)            │
│                             │
│    mainContent              │
│                             │
└─────────────────────────────┘
BottomNav (显示)


只有 mainContent 区域可以滚动
